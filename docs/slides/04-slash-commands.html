What are slash commands?

Not natural language. Direct, unambiguous instructions.
✓
Instant & unambiguous
No prompt parsing — Claude executes the command directly.
✓
Autocomplete
Type / and tab through every available command.
✓
Built-in vs custom
Today is the built-in set. Writing your own comes in Video 12.
terminal
~/project $ claude
> /
  /clear        /compact
  /model        /resume
  /status       /cost
  /permissions  /review

The daily drivers

Five commands you'll reach for in almost every session.
CommandWhat it doesWhen to use it
/clearWipes the conversation contextStarting a new, unrelated task
/compactSummarizes & trims historyContext window is filling up
/modelSwitches model for this sessionNeed more or less power temporarily
/resumeReopens a previous sessionContinuing yesterday's work
/helpLists every available commandYou forgot a command name

Context & diagnostics

Know what's happening under the hood before it surprises you.
01
/status
Session health, active permission mode, and current model at a glance.
02
/cost
Token spend so far this session — check it before a long run finishes.
03
/doctor
Checks your install and configuration for problems.
04
/permissions
Review and edit tool permission rules live, without leaving the session.

Review & collaboration

Commands that connect Claude Code to your team's workflow.
01
/review
Ask Claude to review a diff or an open pull request before you merge.
02
/pr-comments
Pulls existing PR review comments in as context for the session.
03
/bug
Reports an issue with Claude Code directly from where you hit it.

Common mistakes

 

Do this

  • /clear between unrelated tasks
  • Check /cost on long sessions
  • Run /help when a command name slips your mind

Not this

  • Never clearing context across weeks of work
  • Ignoring permission prompts instead of reading /permissions
  • Guessing at command names instead of autocompleting

Built-in today. Custom next.

 

These built-in commands cover the mechanics of running a session well. But the real leverage comes when you encode your own repeated workflows as commands — that's the entire subject of Video 12.

Key takeaways

 
✓

Slash commands are direct instructions, not prose — faster and unambiguous.

✓

/clear, /compact, /model, /resume, and /help cover most day-to-day needs.

✓

/status, /cost, /doctor, and /permissions keep you informed instead of surprised.

✓

Build the habit loop: clear starts, compact near limits, cost checks before you wrap up.
