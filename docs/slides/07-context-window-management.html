What is the context window?

Finite. Not magic. Not infinite.
✓
Everything counts
Every message, every file read, every tool result adds to it.
✓
It's finite
There's a hard limit — it doesn't scroll on forever for free.
✓
Treat it like RAM
Managed deliberately, not filled carelessly and forgotten about.

What fills it up

 
01
Conversation history
Every prompt you send and every response Claude gives back.
02
File reads
The full contents of every file Claude opens to look at.
03
Tool outputs
Command results, search results, diffs — all of it counts.
04
Long correction chains
"No, not like that" repeated ten times adds up fast.

What happens when it saturates

 
Context fills silently
→
Claude starts "forgetting" decisions made earlier in the session.
No one notices yet
→
Code quietly starts contradicting itself — no error, just drift.
Caught early with /status
→
You compact or clear before quality ever degrades.

The three commands

 
CommandWhat it doesWhen to use it
/statusShows current context usageCheck it periodically, not just when worried
/compactSummarizes & trims historyApproaching the limit, need continuity
/clearWipes everythingSwitching to a fully unrelated task

Common mistakes

 

Do this

  • Check /status periodically, not just when things feel off
  • Compact before you hit the wall, not after
  • Clear between genuinely unrelated tasks

Not this

  • Running a single session for days without ever checking usage
  • Waiting until Claude visibly contradicts itself to act
  • Re-reading the same large files over and over instead of referencing them once

The foundation for what's next

 

Managing context well isn't just about one long session — it's the foundation for everything that scales beyond it. Skills and SubAgents, coming up next, both exist specifically to keep your context window clean.

Key takeaways

 
✓

The context window is finite — conversation, file reads, and tool outputs all consume it.

✓

Saturation causes silent drift before it causes visible errors — catch it early.

✓

/status to check, /compact to trim with continuity, /clear to wipe fully.

✓

Good context hygiene is what makes Skills and SubAgents effective later in the series.
